Image and Audio

Image

This part will help to get use to image and audio elements in HTML. To imbed an image or a video into a webpage, you can use this following syntax:

  • For images: <img src="image.jpg" alt="Description">
  • For audio: <audio controls><source src="audio.mp3" type="audio/mpeg"></audio>

Each of these elements has its own attributes, which can be used to customize the appearance and behavior of the media.
